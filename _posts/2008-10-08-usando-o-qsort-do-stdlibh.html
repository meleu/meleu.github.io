--- layout: post title: usando o qsort() do stdlib.h date: '2008-10-08T02:11:00.003-03:00' author: Meleu Zord tags: - C - programacao modified_time: '2008-10-28T15:51:32.554-02:00' blogger_id: tag:blogger.com,1999:blog-1613465446497619833.post-6424992234613074085 blogger_orig_url: http://mdicas.blogspot.com/2008/10/usando-o-qsort-do-stdlibh.html --- O protótipo é o seguinte:

#include <stdlib.h>

void qsort(void *base, size_t nmemb, size_t size, int (*compare)(cont void *, const void*));


Criei uma função compara() para comparar dois inteiros:

int compara(int const *a, int const *b) {
    return *a - *b;
}


e usei o seguinte comando para ordenar um array de inteiros lista[] com 10 elementos.

qsort(lista, 10, sizeof(int), &compara);


Pronto! Agora lista já está ordenada.
